  • Comentários

    • Para comentar no código usa-se // para uma linha ou /* para abrir e */ para fechar um comentário maior.

    Para inserir texto é possível utilizar aspas duplas(""), apostrófo('') e a crase(``). Os dois primeiros tipos não tem diferença, porém o último permite a manipulação de variáveis no texto sem a necessidade de concatenar com o +.

  • Variáveis

    • Identificadores

    Regras para nomeação:

    • Podem começar com letra, $ ou _;
    • Não podem começar com números;
    • É possível usar letras ou números;
    • É possível usar acentos e símbolos;
    • Não podem conter espaços;
    • Não podem ser palavras reservadas.

    Inicialização

    Diferença entre var, let e const:

    • A utilização var cria a variável em todo o programa.
    • A utilização let cria a variável somente no escopo do bloco utilizado.
    • A utilização const cria uma constante invariável.

    Dicas

    • Maiúsculas e minúsculas fazem diferença.
    • Escolha nomes coerentes.
